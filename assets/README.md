## Optimización de Imágenes

Antes de subir las imágenes, optimízalas para web:
- Formato: JPG para fotos, PNG para mockups con transparencia
- Calidad: 80-85% para JPG
- Tamaño máximo: 500KB por imagen
- Herramientas: TinyPNG.com, Squoosh.app
